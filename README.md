# 📋 Smart Task Manager

A modern **Smart Task Manager Web Application** built using **HTML, CSS, and JavaScript** that helps users efficiently organize and manage their daily tasks.

The application allows users to add tasks, set priorities, track progress, and filter tasks based on their completion status.

🔗 **Live Demo**
https://tanushwi.github.io/smart-task-manager/

---

## 🚀 Features

* ➕ Add new tasks
* 📅 Set due date for tasks
* ⭐ Task priority (High / Medium / Low)
* 🔍 Search tasks instantly
* 📂 Filter tasks

  * All Tasks
  * Pending Tasks
  * Completed Tasks
* ✔ Mark tasks as completed using checkbox
* 🗑 Delete tasks
* 📊 Progress bar to track task completion
* 🌙 Dark mode toggle
* 💾 LocalStorage support (tasks remain saved after page refresh)
* 📱 Responsive design

## ⚙ How It Works

1. Users can **add a task** along with a due date and priority.
2. Tasks are stored in the browser using **LocalStorage**, so they remain saved after refreshing the page.
3. Users can **mark tasks as completed** using a checkbox.
4. The **filter buttons** allow users to view:

   * All tasks
   * Only pending tasks
   * Only completed tasks
5. The **search feature** helps quickly find tasks.
6. The **progress bar** dynamically updates based on the number of completed tasks.
